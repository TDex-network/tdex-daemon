Go implementation of the TDex Daemon


## Development


* Build

```sh
# Build for Linux
$ make build-linux

# Build for ARM
$ make build-arm

# Build for Mac
$ make build-mac
```

* Run

```sh
./build/tdexd-<os>-<arch>
```
